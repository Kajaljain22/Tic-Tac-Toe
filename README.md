# Tic-Tac-Toe
Tic-tac-toe, or noughts and crosses is a game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner.
It is built using Unity and C#
